## Dependences

- Node 16.14
- Yarn 1.22.19

## Envs

```bash
$ mv .env.example .env
```
## Installation using Docker

```bash
$ docker compose up --build --force-recreate
```

## Installation with Yarn

```bash
# development
$ yarn

# watch mode
$ yarn start:dev
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
